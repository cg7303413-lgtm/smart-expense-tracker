Smart Expense Tracker

A modern web application that helps users manage their daily expenses, track spending habits, and maintain financial discipline.

Features

- Add and delete expenses
- Track income and expenses
- Monthly budget management
- Budget warning alerts
- Expense analytics and charts
- Local Storage support
- PDF/CSV export
- Responsive design for mobile and desktop
- Clean and user-friendly interface

Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js

How to Use

1. Add your income and expenses.
2. Set a monthly budget.
3. Track your spending through charts and analytics.
4. Export your data when needed.

Future Improvements

- User authentication (Login/Signup)
- Cloud database integration
- AI-based spending insights
- Dark mode support
